# Pirus — концепт редизайна сайта

Концепт нового сайта агентства **Pirus** (SEO и разработка сайтов, Минск).
Все файлы самостоятельные — открываются двойным кликом, без установки.

## Страницы
- `index.html` — главная страница (полный концепт)
- `cube.html` — герой с 3D-кубом (логотип-кубик, летит за мышкой)
- `hero-white.html`, `hero-dark.html` — варианты первого экрана (белый / тёмный)
- `variant-a.html` … `variant-d.html` — ранние варианты моушена
- `archive-index.html` — самый первый черновик

## Бренд
- Шрифт: **Golos Text**
- Цвета: `#69dcdc` (бирюза), `#377b88` (тёмная бирюза), `#0d1e2e` (тёмно-синий текст)
